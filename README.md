# 🚀 AI & Machine Learning Internship – Phase 2

This repository contains 5 AI/ML projects completed as part of the internship program. Each project demonstrates practical applications of Machine Learning, Deep Learning, NLP, and Generative AI.

---

## 📂 Projects

### 1️⃣ News Topic Classifier using BERT

A deep learning NLP project that classifies news articles into multiple categories using the BERT transformer model.

**Key Features**

* BERT-based text classification
* News category prediction
* Model evaluation and inference
* Streamlit web application

---

### 2️⃣ Housing Price Prediction (Multimodal AI)

A machine learning system that predicts house prices using structured property features and image data.

**Key Features**

* Multimodal learning
* Image + tabular data processing
* Price prediction model
* Model evaluation

---

### 3️⃣ Customer Churn Prediction Pipeline

An end-to-end machine learning pipeline that predicts whether customers are likely to leave a service.

**Key Features**

* Data preprocessing
* Feature engineering
* Model training and evaluation
* Saved prediction pipeline

---

### 4️⃣ RAG Chatbot

A Retrieval-Augmented Generation chatbot that answers questions from custom documents using vector search and LLMs.

**Key Features**

* Document ingestion
* Vector database retrieval
* Context-aware responses
* Streamlit interface

---

### 5️⃣ Auto Tagging Support Tickets

An NLP-based system that automatically categorizes customer support tickets.

**Key Features**

* Zero-shot text classification
* Automated ticket routing
* Support ticket categorization
* NLP workflow

---
